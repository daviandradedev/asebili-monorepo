# Demo Data

New instructor accounts without classes or activities receive seeded demo content on the first dashboard visit.

- 2 classes with generated access codes
- 4 activities across quiz, memory, and matching formats
- Public sample MP4 videos
- Student leaderboard entries with quiz answer details

Class codes are shown in the dashboard class list. Use one of those codes in the Expo mobile app while `EXPO_PUBLIC_API_URL` points to the running web API.

To recreate the seeded experience, use a new instructor account or clear that instructor's classes and activities in the database.
